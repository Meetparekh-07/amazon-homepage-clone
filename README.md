# 🛒 Amazon Homepage Clone

A static clone of the Amazon homepage built using **HTML** and **CSS**.  
This project focuses on replicating the layout, design, and user interface of a real-world e-commerce platform for learning and practice purposes.

---

## 🚀 Features

- 🧭 Navigation bar similar to Amazon  
- 🔍 Search bar UI  
- 📦 Product sections & cards  
- 🖼️ Banner / hero section  
- 📱 Basic responsive layout  
- 🎨 Clean and structured UI design  

---

## 🛠️ Technologies Used

- **HTML5** – for structuring the webpage  
- **CSS3** – for styling and layout
